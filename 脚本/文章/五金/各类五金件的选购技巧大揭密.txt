	<span style="font-size:16px;font-family:SimSun;">　　五金件在家装费用中约占3%-5%，比例虽小，却不可忽视。五金件按使用功能分为普通五金类和特殊五金类。普通五金类按设置方式分为锁具、铰链、滑轨、拉手、吊轮、钉、板托、门吸、衣钩、碰珠、碰卡、板销等。特殊五金类按设置方式分为浴室五金类和厨房挂件类。本文只介绍普通五金类，特殊五金类后期待续。</span><br />

<span style="font-size:16px;font-family:SimSun;">　　（一）锁具</span><br />
<span style="font-size:16px;font-family:SimSun;">　　1、特点&nbsp;</span><br />
<span style="font-size:16px;font-family:SimSun;">　　如今市面上的锁具种类繁多，就门锁而言，按用途分门户锁、卧室锁、通道锁、浴室锁等；按形状分球型锁、执手锁、插芯锁、呆锁等。球型锁和执手锁具有锁、拉手和碰球三种功能。呆锁要另外安装门拉手。</span><br />
<span style="font-size:16px;font-family:SimSun;">　　</span><span style="font-size:16px;font-family:SimSun;">①户门锁：起保险安全作用，所以又称保险锁或防盗锁。</span><br />
<span style="font-size:16px;font-family:SimSun;">　　②通道锁：起门的拉手和撞珠的作用，没有保险功能，适用于厨房、过厅、客厅、餐厅及儿童间的门锁。</span><br />
<span style="font-size:16px;font-family:SimSun;">　　③浴室锁：特点是在里面能锁住，在门外用钥匙能打开，适用于卫生间或浴室。</span><br />
<span style="font-size:16px;font-family:SimSun;">　　④卧室锁：在里面锁上保险，外面必须用钥匙开启，适用于卧室及阳台门。</span><br />
<span style="font-size:16px;font-family:SimSun;">　　用于锁具的材料主要有铜、不锈钢、锌合金、钢铁、铝或铝合金等。</span><br />
<span style="font-size:16px;font-family:SimSun;">　　</span><span style="font-size:16px;font-family:SimSun;">①铜：是使用最广泛的锁具材料之一，其机械性能好，耐腐蚀和加工性能都不错，且色泽艳丽，特别是铜锻造的把手及其他锁具装饰件，表面平整、密度好、无气孔、砂眼。</span><br />
<span style="font-size:16px;font-family:SimSun;">　　②不锈钢：耐用，越用越光亮。其强度好、耐腐蚀性强、颜色不变。</span><br />
<span style="font-size:16px;font-family:SimSun;">　　③锌合金材料：其强度和防锈能力较差，优点是易于做成复杂图案的零件，特别是压力铸造。市面所见的较复杂图案的锁多是锌合金的。</span><br />
<span style="font-size:16px;font-family:SimSun;">　　④钢铁：强度较好，成本较低，一般外加镀层，但易生锈。</span><br />
<span style="font-size:16px;font-family:SimSun;">　　⑤铝或铝合金：普通铝合金(航空航天用除外)质软且轻、材料强度较底，但易于加工成形。</span><br />
<span style="font-size:16px;font-family:SimSun;">　　</span><span style="font-size:16px;font-family:SimSun;">2、选择</span><br />
<span style="font-size:16px;font-family:SimSun;">　　消费者要选择铜芯琐，但要注意有的琐芯是塑料的，只在芯头处包了一层铜皮的“铜芯”琐。全铜镀镍和铁镀镍琐外观几乎一样，但价格差别很大，一般全铜琐带静音，铁琐不带静音，细看全铜琐做工精细，铁琐做工较粗糙。球型琐安装时如偏芯等装不到位，使用中极易出毛病。现在有一种反手琐，手柄轻轻一抬，就能琐死，非常方便，但容易把钥匙琐在屋内。</span><br />
<span style="font-size:16px;font-family:SimSun;">　　现在，卖得最快的是执手锁，球形锁过时了，一般不用了。相对而言，转动琐具时，声音越小的越好，手柄转一点，琐舌就应跟进一点。从钥匙形状来讲，“坑”状钥匙琐具比“牙”状钥匙琐具的安全性高些，但“坑”状钥匙丢了必须要到原厂配。琐具的方头舌数量越多，安全性越好。消费者选择琐具时要注意以下几点。</span><br />
<span style="font-size:16px;font-family:SimSun;">　　①应选择知名度高、质量稳定、售后服务好的企业的产品。</span><br />
<span style="font-size:16px;font-family:SimSun;">　　②先检查所购产品包装的标志是否齐全(包括产品的执行标准、等级、生产企业名称、地址、生产日期)，包装物是否牢固，说明书内容与产品是否相符，慎防有夸大而与事实不符现象。</span><br />
<span style="font-size:16px;font-family:SimSun;">　　</span><span style="font-size:16px;font-family:SimSun;">③观察产品外观质量情况，包括锁头、锁体、锁舌、执手与覆板部件及有关配套件是否齐全，电镀件、喷漆件表面色泽是否鲜艳、均匀，有无生锈、氧化迹象及破损。</span><br />
<span style="font-size:16px;font-family:SimSun;">　　④检查产品的使用功能是否可靠、灵活，应选择两把以上的产品进行对照检查，特别是购买双向锁头产品时，必须用全部钥匙分别对内、外锁头进行试开关。还应检查产品的保险机构情况，建议每把锁至少试三次以上。</span><br />
<span style="font-size:16px;font-family:SimSun;">　　⑤按使用环境选择琐具，如干湿度、门的结构、厚度、左开门还是右开门，内开门还是外开门等。</span><br />
<span style="font-size:16px;font-family:SimSun;">　　⑥互开率是锁和钥匙的互开比例，一把钥匙打开的锁越少，证明这把锁的安全可靠性越强。消费者在选购时要尽量去挑选钥匙牙花数多的锁，因为钥匙的牙花数越多，差异性越大，锁的互开率就越低。</span><br />
<span style="font-size:16px;font-family:SimSun;">　　（二）拉手</span><br />
<span style="font-size:16px;font-family:SimSun;">　　</span><span style="font-size:16px;font-family:SimSun;">1、特点</span><br />
<span style="font-size:16px;font-family:SimSun;">　　拉手从材质来分有单一金属、合金、塑料、陶瓷、玻璃等；从外形来分有管形、条形、球形及各种几何形状等；从式样来分又有单条式、双头式、外露式、封闭式等；从款式来分既有比较前卫或休闲的，也有以绳索或吊珠式的怀古拉手。</span><br />
<span style="font-size:16px;font-family:SimSun;">　　2、选择</span><br />
<span style="font-size:16px;font-family:SimSun;">　　拉手的花样太多，款式也不断翻新，拉手的选择也各有所好。从材质上讲，全铜、全不锈钢的较好，合金、电镀的较差，塑料的正濒临淘汰。</span><br />
<span style="font-size:16px;font-family:SimSun;">　　拉手有用螺钉和胶粘两种固定方式，一般用螺钉固定的结实，胶粘的不实用。全不锈钢拉手和镀不锈钢拉手外观差不多，可用磁铁试出来。拉手在选配时必须注意家具的款式、功能和场所，一般来说，拉手与家具的关系大致有两种处理原则，要么是醒目，要么是隐蔽。</span><br />
<span style="font-size:16px;font-family:SimSun;">　　</span><span style="font-size:16px;font-family:SimSun;">玄关是居室里重点装饰的部位之一，玄关柜的拉手可以强调它的装饰性。一般来说，对称式的装饰门上安装两个豪华漂亮的拉手，而鞋柜则应挑选色泽与板面接近的单头式拉手。这些以使用功能为主的家具，其拉手应该具有隐性，以不妨碍主人使用为妥。食品装饰柜的拉手可以与其自身较为抢眼的格调相适应，选购具有光泽并与家具色泽有反差的双头式拉手。电视机柜的拉手可以考虑选择与电器件或电视柜台面石材色泽相近，如黑色、灰色、深绿色、亚金色的外露式拉手。其理由是这些地方的柜门开启的频率较低，再就是为了保证人的走动不至于发生牵扯。书房或工作室的家具可以模仿写字楼的做法，挑选简洁方正的拉手。卫浴间的柜门不多，适宜挑选微型单头圆球式的陶瓷或有机玻璃拉手，其色泽或材质应与柜体相近。</span><br />
<span style="font-size:16px;font-family:SimSun;">　　（三）铰链</span><br />
<span style="font-size:16px;font-family:SimSun;">　　1、特点</span><br />
<span style="font-size:16px;font-family:SimSun;">　　铰链(合页)按用途分平板、弹簧(烟斗)、自关、玻璃铰链等；按材质分不锈钢、铜、钢加镀层、铸铁铰链等。</span><br />
<span style="font-size:16px;font-family:SimSun;">　　平板铰链多用于房间木门，弹簧铰链多用于柜门，自关铰链多用于卫生间，玻璃铰链用于玻璃门。</span><br />
<span style="font-size:16px;font-family:SimSun;">　　</span><span style="font-size:16px;font-family:SimSun;">2、选择</span><br />
<span style="font-size:16px;font-family:SimSun;">　　平板铰链一般有四个轴承，平板铰链的好坏主要取决于轴承的质量，相对而言，轴承直径越大越好，壁板越厚越好，手平拿铰链一片，让另一片自由下滑，匀速且慢的较好。弹簧(烟斗)铰链有全盖、半盖、不盖之分，用于柜门与柜体的三种不同连接方式。选择弹簧铰链主要是认准品牌，小品牌铰链的弹簧片大部分容易老化、疲劳，致使柜门下垂。不锈钢和钢板铰链的壁板较薄，但韧性好，不易断裂。铸铁铰链虽然壁板较厚，但易断裂，有的商家故意蒙骗消费者，说壁板越厚越贵，其实材质不同。选择弹簧铰链时，还要注意铰链上不要缺少调节螺丝，因为此螺丝丢了不易配，没有单卖的。</span><br />
<span style="font-size:16px;font-family:SimSun;">　　平板铰链壁板的厚度与门扇的重量有关，一般40公斤以上的门扇，平板铰链壁板的厚度要在3.2 mm 以上。大部分10元一付的平板铰链没有全轴承，一般是两个真的两个假的。弹簧铰链的价格差别较大，知名品牌的壁板较厚，做工精细；小品牌的壁板较薄，做工粗糙。玻璃铰链有中间轴和上下轴之分，中间轴的需要打孔，用的越来越少；上下轴的不需要打孔，是大部分消费者的选择。玻璃铰链配套有玻璃钉和玻璃托，有全不锈钢和镀不锈钢两种，全不锈钢的较贵。</span><br />
<span style="font-size:16px;font-family:SimSun;">　　（四）钉</span><br />
<span style="font-size:16px;font-family:SimSun;">　　1、特点</span><br />
<span style="font-size:16px;font-family:SimSun;">　　</span><span style="font-size:16px;font-family:SimSun;">家装中用到的钉主要有钢钉、气钉、气钢钉、水泥钉、自攻钉、蚊钉、特种钉等。</span><br />
<span style="font-size:16px;font-family:SimSun;">　　①钢钉是最普通的钉类，没有专用工具，主要用于手工钉的一些木质架子等。</span><br />
<span style="font-size:16px;font-family:SimSun;">　　②气钉有专用工具，主要用于木板、木龙骨等。</span><br />
<span style="font-size:16px;font-family:SimSun;">　　③气钢钉用于木龙骨与墙体连接处，有专用工具，与气钉相比，压力相对较大，气钢钉主要钉水泥墙。</span><br />
<span style="font-size:16px;font-family:SimSun;">　　④水泥钉没有专用工具，主要是于手工钉水泥墙等。</span><br />
<span style="font-size:16px;font-family:SimSun;">　　</span><span style="font-size:16px;font-family:SimSun;">⑤蚊钉全是铜质，钉孔小，主要用于装饰面板。</span><br />
<span style="font-size:16px;font-family:SimSun;">　　⑥特种钉带螺纹，主要用于钉踢脚线，螺纹结构不易松动。</span><br />
<span style="font-size:16px;font-family:SimSun;">　　2、选择</span><br />
<span style="font-size:16px;font-family:SimSun;">　　气钉的用量最大。气钉按盘卖，选择时主要看其排列是否整齐，长短是否一致。需要注意的是，由于每盒气钉的数量较多，有的气钉实际数量比标称数量少。</span><br />
<span style="font-size:16px;font-family:SimSun;">　　（五）吊轮</span><br />
<span style="font-size:16px;font-family:SimSun;">　　</span><span style="font-size:16px;font-family:SimSun;">1、特点</span><br />
<span style="font-size:16px;font-family:SimSun;">　　吊轮用于阳台、厨房、餐厅的推拉门，有进口吊轮和国产吊轮。大部分进口吊轮只是轴承是进口的，其他部件是国产的。</span><br />
<span style="font-size:16px;font-family:SimSun;">　　2、选择</span><br />
<span style="font-size:16px;font-family:SimSun;">　　进口吊轮因为采用的是进口轴承，大部分是超静音的，使用寿命在15-20年左右，价格在80元左右。相比之下，国产吊轮的声音稍大，使用寿命在5-10年左右，带顶轮和侧轮的价格在45元左右，不带顶轮和侧轮的价格在20元左右。消费者选择吊轮时，可把吊轮放在玻璃柜台上，用手按住来回推拉，多试几个样品，选择手感好、无噪音的。</span>
