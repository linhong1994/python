<p>
	<span style="font-size:16px;font-family:SimSun;">　　随着大城市住房房价的不断升高，小户型住房成了很多人定居城市生活的首选。狭小的空间要想完全的利用起来就越发的困难，但是这并不能难倒我们消费者，新型的装修理念开始走进他们的脑海。如何合理利用小空间，将小的空间"放大化"也成了很多新房装修人关注的话题。那么，对于小空间的卫浴间，我们有什么方法能将它变大呢？</span> 
</p>
<span style="font-size:16px;font-family:SimSun;"> 　　</span><br />
<span style="font-size:16px;font-family:SimSun;"> 　　向上推进</span><br />
<span style="font-size:16px;font-family:SimSun;"> 　　</span><br />
<span style="font-size:16px;font-family:SimSun;"> 　　在我们传统的卫浴间设计中，马桶、洗脸盆、浴室柜等等都是立体放置，大大占据了我们下脚的空间。鉴于这种情况，我们在小空间安装这些洁具用品可利用整体的空间，如：利用坐便器和盥洗盆上方的空间安装一排浴柜，可用来收纳种类繁多的洗浴用品；另外，也可以选择挂墙式的浴室柜，这样所占据的空间就会大大减少，而且还可以收纳更多的东西。</span><br />
<span style="font-size:16px;font-family:SimSun;"> 　　</span><br />
<span style="font-size:16px;font-family:SimSun;"> 　　向下延伸</span><br />
<span style="font-size:16px;font-family:SimSun;"> 　　</span><br />
<span style="font-size:16px;font-family:SimSun;"> 　　洗手池下方往往会安装管道，不仅影响美观而且占据空间。我们可将这部分做成浴柜，收纳一些平常使用的小物品，如洗发水、沐浴露等，或是只拉上布帘，储藏一些盒子和小箱子之类的东西。</span><br />
<span style="font-size:16px;font-family:SimSun;"> 　　</span><br />
<span style="font-size:16px;font-family:SimSun;"> 　　墙面利用</span><br />
<span style="font-size:16px;font-family:SimSun;"> 　　</span><br />
<span style="font-size:16px;font-family:SimSun;"> 　　面积小就要充分利用空间，墙面也不例外。我们在装修卫浴间时，千万不能让墙面空着，可以把毛巾、手纸、电吹风这些小玩意儿统统请上墙面，也可以在墙上悬挂一块大些的玻璃，这样就可放大内部空间。除此之外，还可将储物柜镶在墙内，用时拉出，不用时也可以隐藏起来。</span><br />
<span style="font-size:16px;font-family:SimSun;"> 　　</span><br />
<span style="font-size:16px;font-family:SimSun;"> 　　色彩搭配</span><br />
<span style="font-size:16px;font-family:SimSun;"> 　　</span><br />
<span style="font-size:16px;font-family:SimSun;"> 　　为避免视觉上的疲劳感和空间上的拥挤感，应选择清洁而明快的色彩为主要背景色，对缺乏透明度与纯净感的色彩要敬而远之。在搭配上，要强调统一性和融合感，过于鲜艳夺目的色彩不宜大面积使用，以减少色彩对人心理的冲击与压力。如果几种颜色搭配，则最好采用大色块装饰，如白色瓷砖墙壁、浴池和洗脸池与红色的浴帘，黑色的天棚、毛巾架映衬，这样的搭配在视觉上不拥挤。</span><br />
<span style="font-size:16px;font-family:SimSun;"> 　　</span><br />
<span style="font-size:16px;font-family:SimSun;"> 　　巧用配件</span><br />
<span style="font-size:16px;font-family:SimSun;"> 　　</span><br />
<span style="font-size:16px;font-family:SimSun;"> 　　小空间当然要选择尺寸较小的洗脸盆或抽水马桶，尽量不采用浴缸。在合理的干湿分区基础上，达到浴室空间的充分利用。并且一般选用白色、线条简单的洁具和金属质感的配件进行搭配。像淋浴房，我们可用玻璃挡板或挂帘来遮挡。</span>
