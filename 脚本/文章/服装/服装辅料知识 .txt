</br>一、服装辅料知识之标识 </br>　　服装标识的设立的初衷是防止人们在购买服饰时，将本品牌的服饰与其他品牌服饰混淆而设计的一种区别符号。而现在随着企业处于对服装文化的重视，服装标识不再仅仅是为了区别，更多的是考虑到将本企业的文化内涵向人们传播出去。所以很大程度上，服装标识成了一种无形资产的表达方式，这就是一个品牌的文化精髓。一个品牌名声响亮与否都随时决定着一个消费者对服装的选择，所以标识发展到今天已经成了一个企业可持续发展的保障。越来越多的企业加大对对标识的投资，目的就是为了打造品牌“航母”，筑起企业“文化长城”，同时也成了企业与消费者连接的精神纽带。特别在当今这样市场激烈竞争的环境，一个企业成功的品牌战略将是取胜的无冕之王。 </br>标识名目繁多，大概如下面几种：</br></br>　　1、商标。通俗说法为布标，织唛。多数用于领标和其它装饰的地方。 </br></br>　　2、吊牌。又称为牌仔，纸牌。其功能主要是诉述品牌的特点和服饰的成分。</br></br>　　3、胶章。滴塑章的正称。胸标等地方常见。 </br></br>　　4、PVC章。一般由塑料制成，塑料的好处就是可以随意更改形象，灵活多变。 </br></br>　　5、反光章及反光材料。主要特点是反光剧烈，是装饰不可多得的材料。 </br></br>　　6、三维章。呈一种反光晶格立体图案，要经过特种印刷机八次印刷，制作过程比较复杂。 </br></br>　　7、吊粒。谓称吊钟。常见于高品位西装、时装。 </br></br>　　8、织带。有台湾木梭机织带、反光织带、尼龙、全棉人字带等。 </br></br>二、服装辅料知识之商标 </br></br>　　制作工艺流程类似与织带，就是用纱线编织而成，主要是黑色或白色。图案、文字用纬纱构成。（如有其它特种要求而将经纱染成其它颜色的，称染经商标。） </br></br>　　商标分类方法很特别也很简单，主要是按照所印制的机器不同和制作工艺流程不同来划分。主要分为以下几种大类。 </br></br>　　1、西德机商标 </br></br>　　特点：商标沿经类似于一匹布的，出来后再进行切割，所以这种切割办法又可以叫切边机。 </br>　　等上面功课完成后再按其纹路不同，之后可以分为两大类，平面（平纹）和缎面（斜纹）商标。其中平面又可分一般平面，双面，重梭等多种。当然，按所用经纱不同可为黑平、白平和黑缎、白缎。 </br></br>　　2、木梭机商标(其中最主要是台湾和日本木梭机) </br></br>　　特点：商标从机器上编织出来出来后就是完整的一条，故其边为包边，而纬纱从头到尾如出一辙，颜色样式一模一样，摸起来特别细腻柔软。但都有几点不足之处，例如生产效率低、使用的颜色局限在五种以下。跟西德机大同小异。 </br></br>　　3、钩针机商标。 </br></br>　　特点雷同木梭机商标，但两者之间有很大的差别，因为这种不是梭机，而是针织，纬纱多为为紧、强。常用来作烫色标，利用经、纬纱不同的特点使底板经高温转印而上色。商标成本较高。 </br></br>三、服装辅料知识之吊牌 </br></br>　　1、常见的吊牌为柯式印刷纸类制品。而丝网印刷的塑料制品等比较少见和少用。 </br></br>　　2、柯式印刷原理颇为简单，品质的高低绝对取决于机器设备的好坏；纸张加工较为灵活，可以凭着设计师的喜好而进行各种处理，所以一般情况下都能将设计师的原创灵魂灵活地表现出来。 </br></br>　　3、每个东西的制作都有着基本的工序，吊牌设计也不例外，但必须要仔细安排好考虑好妥当安排好每一个步骤。</br></br>　　基本的工序只要分为以下几大类别 </br></br>　　图案（画稿）—制版—印刷—用纸—后加工 （顺序从前到后）</br></br>四、服装辅料知识之关于胶章 </br></br>　　胶章，又称滴塑章，或者PVC章，通过雕模，高温定型等工艺，产品特点为表达方式灵活，富有立体效果。为近年很多企业都是通过此种品牌表达。 </br></br>五、服装辅料知识之PVC LOGO </br></br>　　PVC LOGO，利用人造皮革和各种塑料的丰富可塑性，加上丝网印刷和高频热压等工艺处理，得到各种材料柔和的灵活造型。 </br></br>六、服装辅料知识之反光材料。 </br></br>　　反光材料，利用反光晶格反光涂料，加强光线的反射效果，在强光或夜视条件下，能使标志物格外显目，容易使人在夜里比较容易分辨事物。
